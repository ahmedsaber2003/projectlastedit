Lesson 3: Boolean Expressions and Conditions

This lesson introduces Boolean expressions in PascalABC.NET. Boolean expressions are used to compare values and return one of two possible results: True or False.

1. What Is a Boolean Expression?

A Boolean expression is an expression that produces a logical value: True or False. Boolean expressions are often built using comparison operators.

For example:

a > 5

This expression checks whether the value of a is greater than 5.

Example: condition is True

begin
 var a := 6;
 writeln(a > 5);
end.

Output:

True

Example: condition is False

begin
 var a := 5;
 writeln(a > 5);
end.

Output:

False

2. Comparison Operators

Comparison operators are used to compare numbers and other values. The result of every comparison is a Boolean value.

  • > — greater than
  • >= — greater than or equal to
  • < — less than
  • <= — less than or equal to
  • = — equal to
  • <> — not equal to

Example with greater than or equal

begin
 var a := 5;
 writeln(a >= 5);
end.

Output:

True

3. Logical Operations

Conditions can be combined using logical operations: not, and, and or.

  • not changes True to False and False to True;
  • and returns True only when both conditions are True;
  • or returns True when at least one condition is True.

When a condition consists of several parts, each condition should be written in round brackets.

Example with or

begin
 var year := 19;
 writeln((year < 20) or (year > 18));
end.

Output:

True

Example with not and and

begin
 var a := 5;
 writeln((not (a < 4)) and (7 > 5));
end.

Output:

True

4. Printing Boolean Results

In this lesson, conditions are checked without using the if statement. Instead, the result of the condition can be printed directly using writeln, write, or print.

Example

begin
 var a := 5;
 write(a >= 5);
end.

Output:

True

5. The odd Function

PascalABC.NET provides the standard function odd. It returns True if its argument is an odd integer and False otherwise.

Example

begin
 print(odd(5));
 print(odd(6));
end.

Result:

TrueFalse

The function odd is useful when checking whether at least one of two numbers is odd.

Example: at least one number is odd

begin
 var a := ReadInteger('Enter a: ');
 var b := ReadInteger('Enter b: ');
 print(odd(a) or odd(b));
end.

6. Double Inequality

A double inequality such as A < B < C should be written as two separate comparisons connected by and. 

(A < B) and (B < C)

Example

begin
 var A := ReadInteger('Enter A: ');
 var B := ReadInteger('Enter B: ');
 var C := ReadInteger('Enter C: ');
 print((A < B) and (B < C));
end.

7. Working with Digits and Conditions

Boolean expressions can also be combined with digit extraction. For example, for a three-digit number, we can check whether the first digit is less than the second and third digits.

Example

begin
 var n := ReadInteger('Enter a three-digit number: ');
 var x := Abs(n);
 var hundreds := x div 100;
 var tens := (x mod 100) div 10;
 var units := x mod 10;
 print((hundreds < tens) and (hundreds < units));
end.

8. Practice Tasks

  1. Two integers are given. Check whether the first number is greater than the second.
  2. Two integers are given. Check whether the first number is not equal to the second.
  3. Three integers A, B, and C are given. Check the double inequality A < B < C.
  4. A three-digit integer is given. Check whether the first digit is less than the second and third digits.
  5. Two integers are given. Check whether at least one of them is odd.

In these tasks, do not use the if statement. Print only the Boolean result: True or False.

9. Important Notes

  • Boolean expressions return only True or False.
  • Use round brackets when combining several conditions.
  • Use and when all conditions must be true.
  • Use or when at least one condition must be true.
  • Use not to reverse a condition.
  • Use odd(n) to check whether a number is odd.
