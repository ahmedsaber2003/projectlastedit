Lesson 2: Integers, div and mod Operators

This lesson explains integer division, the div operator, the mod operator, and how these operations can be used to extract digits from numbers.

1. Integer Division with div

The div operation performs integer division. It returns only the integer part of the division result. 

10 div 2  = 5
22 div 7  = 3
65 div 10 = 6

The result of N div K shows how many full times K fits inside N.

Example in PascalABC.NET

begin
  var n := 22;
  var k := 7;
  var result := n div k;
  writeln(result);
end.

2. Remainder with mod

The mod operation calculates the remainder after integer division. If a number is divided evenly, the remainder is zero. 

10 mod 2  = 0
22 mod 7  = 1
65 mod 10 = 5

The result of N mod K shows what remains after removing the maximum number of full parts of size K.

Example in PascalABC.NET

begin
  var n := 65;
  var k := 10;
  var result := n mod k;
  writeln(result);
end.

3. Quotient and Remainder Formula

Any integer number can be represented using the quotient and remainder. If: 

d = N div K
m = N mod K

then the original number can be restored using the formula: 

N = d * K + m

Examples

10 div 2 = 5, 10 mod 2 = 0  =>  10 = 5 * 2 + 0
22 div 7 = 3, 22 mod 7 = 1  =>  22 = 3 * 7 + 1
65 div 10 = 6, 65 mod 10 = 5 => 65 = 6 * 10 + 5

PascalABC.NET example

begin
  var n := 22;
  var k := 7;

  var d := n div k;
  var m := n mod k;

  writeln('Quotient = ', d);
  writeln('Remainder = ', m);
  writeln('Check: ', d * k + m);
end.

4. Standard Form of a Number

A number can be separated into digits using powers of 10. This is called the standard form of a number. 

123 = 1 * 100 + 2 * 10 + 3

In this example:

  • 1 is the hundreds digit;
  • 2 is the tens digit;
  • 3 is the units digit.

5. Getting Digits of a Three-Digit Number

For a three-digit number, div and mod can be used to get each digit.

Hundreds digit

123 div 100 = 1

Tens digit

123 mod 100 = 23
23 div 10 = 2

So the complete expression is:

(123 mod 100) div 10

Units digit

123 mod 10 = 3

Full PascalABC.NET example

begin
  var n := 123;

  var hundreds := n div 100;
  var tens := (n mod 100) div 10;
  var units := n mod 10;

  writeln('Hundreds = ', hundreds);
  writeln('Tens = ', tens);
  writeln('Units = ', units);
end.

6. Working with Negative Numbers

Some tasks use negative numbers. In such cases, it is often useful to work with the absolute value of the number when extracting digits. 

begin
  var n := -105;
  var x := Abs(n);

  var hundreds := x div 100;
  var tens := (x mod 100) div 10;
  var units := x mod 10;

  writeln(units, ', ', tens, ', ', hundreds);
end.

7. Common Task Examples

Example 1: centimeters to meters

A distance in centimeters is given. To find the number of full meters, use integer division by 100. 

begin
  var centimeters := ReadInteger('Enter distance in centimeters: ');
  var meters := centimeters div 100;

  writeln('Full meters = ', meters);
end.

Example 2: kilograms to tons

begin
  var kilograms := ReadInteger('Enter mass in kilograms: ');
  var tons := kilograms div 1000;

  writeln('Full tons = ', tons);
end.

Example 3: age in months to years

begin
  var months := ReadInteger('Enter age in months: ');
  var years := months div 12;

  writeln('Full years = ', years);
end.

Example 4: sum and product of digits

begin
  var n := ReadInteger('Enter a two-digit number: ');
  var x := Abs(n);

  var leftDigit := x div 10;
  var rightDigit := x mod 10;

  writeln('Sum = ', leftDigit + rightDigit);
  writeln('Product = ', leftDigit * rightDigit);
end.
