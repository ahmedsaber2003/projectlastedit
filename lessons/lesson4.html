Lesson 4: Condition Operator and IF Statement

This lesson explains conditional branching in PascalABC.NET using the if statement. Conditions allow a program to make decisions and execute different instructions depending on whether a condition is True or False.

1. Short Form of the IF Statement

The short form of the if statement executes one operator only when the condition is true. 

if condition then
 operator

If the condition is True, the operator after then is executed. If the condition is False, the operator is skipped.

Example

begin
 var a := 5;
 if a > 7 then
   print(a - 1);
 print(a);
end.

In this example, a > 7 is false, so print(a - 1) is skipped. The last statement print(a) runs normally.

2. Full Form of the IF Statement

The full form of the if statement uses else. The first operator is executed when the condition is true, and the second operator is executed when the condition is false. 

if condition then
 operator1
else
 operator2

Example

begin
 var a := 5;
 if a > 7 then
   print(a - 1)
 else
   print(a + 1);
 print(a);
end.

Here, a > 7 is false, so the program prints a + 1, which is 6. Then it prints the original value of a.

3. Important Rule: No Semicolon Before else

In PascalABC.NET, you should not write a semicolon before else.

Correct

if a > 7 then
 print(a - 1)
else
 print(a + 1);

Wrong

if a > 7 then
 print(a - 1);
else
 print(a + 1);

The semicolon before else is one of the most common mistakes when learning conditional statements.

4. Several Conditions

A condition may contain more than one logical expression. In this case, each condition should be written in round brackets. 

if (year < 20) or (year > 18) then
 print('condition is true');

The logical operator or returns True if at least one condition is true.

Example with and

begin
 var age := 18;
 if (age >= 18) and (age < 60) then
   print('adult');
end.

5. begin..end Blocks

If you need to execute more than one operator after then or else, use a begin..end block. 

if condition then
begin
 operator1;
 operator2;
end
else
begin
 operator3;
 operator4;
end;

Example

begin
 var a := 10;
 if a > 0 then
 begin
   writeln('Positive number');
   writeln('The value is ', a);
 end
 else
 begin
   writeln('Not positive');
   writeln('The value is ', a);
 end;
end.

6. Odd and Even Numbers

The function odd(n) returns True if n is odd. It can be used together with the if statement. 

begin
 var n := ReadInteger('Enter n: ');
 if odd(n) then
   writeln('odd')
 else
   writeln('even');
end.

To check whether a number is even, you can use not odd(n). 

begin
 var n := ReadInteger('Enter n: ');
 if not odd(n) then
   writeln('even')
 else
   writeln('odd');
end.

7. Changing a Number Using IF

Conditional statements are often used to change the value of a variable only in a certain case.

Example: add 1 if the number is positive

begin
 var n := ReadInteger('Enter n: ');
 if n > 0 then
   n += 1;
 writeln(n);
end.

Example: multiply by 10 if the number is even

begin
 var n := ReadInteger('Enter n: ');
 if not odd(n) then
   n *= 10;
 writeln(n);
end.

8. Multiple Branches

Sometimes a program must choose between several possible results. This can be done using several if statements or else if style conditions.

Example: age category

begin
 var age := ReadInteger('Enter age: ');
 if age >= 18 then
   writeln('you can watch this movie')
 else if age < 10 then
   writeln('you should watch the cartoon')
 else
   writeln('you can take a walk');
end.

Example: student mark

begin
 var mark := ReadInteger('Enter mark: ');
 if mark = 2 then
   writeln('it''s very bad')
 else if mark = 3 then
   writeln('it''s bad')
 else if mark = 4 then
   writeln('it''s good')
 else if mark = 5 then
   writeln('it''s excellent')
 else
   writeln('such marks do not exist');
end.

9. Swapping Values with IF

The if statement can be used before swapping values. For example, if A is greater than B, the program can swap them.

Swap two variables

begin
 var A := ReadInteger('Enter A: ');
 var B := ReadInteger('Enter B: ');
 var temp := A;
 A := B;
 B := temp;
 writeln('A = ', A);
 writeln('B = ', B);
end.

Swap only if A is greater than B

begin
 var A := ReadInteger('Enter A: ');
 var B := ReadInteger('Enter B: ');
 if A > B then
 begin
   var temp := A;
   A := B;
   B := temp;
 end;
 writeln('A = ', A);
 writeln('B = ', B);
end.

10. Common Mistakes

  • Writing a semicolon before else.
  • Forgetting begin..end when several operators are used.
  • Forgetting round brackets in combined conditions.
  • Using = for assignment instead of :=.
  • Forgetting that statements after if run normally.

11. Practice Tasks

  1. If an integer is positive, add 1 to it and output the result.
  2. If an integer is even, multiply it by 10 and output the result.
  3. If an integer is even, multiply it by 3; otherwise divide it by 3.
  4. If an integer is positive, add 1; otherwise subtract 2.
  5. Classify a person's age and print the correct message.
  6. Print a message depending on a student's mark.
  7. Classify the time of day by the entered hour.
  8. If A and B are different, assign their sum to both variables; otherwise assign zero.
  9. Swap the values of A and B.
  10. If A is greater than B, swap their values.
