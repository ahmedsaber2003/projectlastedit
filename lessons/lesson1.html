Lesson 1: Introduction to PascalABC.NET

This lesson introduces the foundations of PascalABC.NET: variables, assignment, arithmetic operations, input, output, formatted printing, and swapping values between variables.

1. Variables and Assigning Values

In PascalABC.NET, variables can be declared inside the program body. You can declare a variable with an explicit type, or let Pascal infer the type from the assigned value.

Example: explicit type

begin
  var n: integer;
  n := 1;
  writeln(n);
end.

Example: inferred type

begin
  var n := 1;
  writeln(n);
end.

2. Arithmetic Operations and Expressions

Pascal supports addition, subtraction, multiplication, and division. You can write them in the standard form or use short operators.

Standard form

begin
  var a := 6;
  a := a + 2;
  a := a - 2;
  a := a * 3;
  a := a / 2;
  writeln(a);
end.

Short form

begin
  var a := 6;
  a += 2;
  a -= 2;
  a *= 3;
  a /= 2;
  writeln(a);
end.

3. Data Input

There are several ways to read values in PascalABC.NET.

Method 1: read

begin
  var n: integer;
  read(n);
  writeln('n = ', n);
end.

Method 2: ReadInteger and ReadReal

begin
  var n := ReadInteger();
  var x := ReadReal();
  writeln(n);
  writeln(x);
end.

Method 3: tuple assignment

begin
  var n1, n2: integer;
  (n1, n2) := (1, 2);
  writeln(n1, ' ', n2);
end.

Method 4: reading two integers at once

begin
  var (n1, n2) := ReadInteger2;
  writeln(n1, ' ', n2);
end.

Prompting the user

begin
  var x := ReadInteger('Enter x: ');
  var y := ReadInteger('Enter y: ');
  var res := x + y;
  writeln('Result = ', res);
end.

4. Data Output

PascalABC.NET can print values using writeln, print, and formatted output.

Using writeln

begin
  var n: integer;
  read(n);
  n := n * n;
  writeln('n = ', n);
end.

Using print

begin
  var n: integer;
  read(n);
  n := n * n;
  print('n = ', n);
end.

Formatted output with WritelnFormat

begin
  var a := 1.2;
  var b := 4;
  var c := a + b;
  WritelnFormat('f({0}, {1}) = {2}', a, b, c);
end.

Formatted output with Print

begin
  var x := ReadInteger('Enter x: ');
  var y := ReadInteger('Enter y: ');
  var res := x + y;
  Print($'Sum of {x} and {y} is {res}');
end.

Field width example

begin
  var x := 6;
  WritelnFormat('x = *{0,5}*', x);
end.

5. Swapping Variable Values

Values can be swapped using a temporary variable or by multiple assignment.

Initial values

var (x, y) := (3, 5);

Method 1: temporary variable

begin
  var (x, y) := (3, 5);
  var t := x;
  x := y;
  y := t;
  writeln('x = ', x, ', y = ', y);
end.

Method 2: multiple assignment

begin
  var (x, y) := (3, 5);
  (x, y) := (y, x);
  writeln('x = ', x, ', y = ', y);
end.
